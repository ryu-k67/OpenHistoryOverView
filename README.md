## 使用予定技術
- フロントエンド：Next.js
- バックエンド&emsp;：Django
- データベース&emsp;：MySQL
- インフラ&emsp;&emsp;&emsp;：Docker

### 概要
- 自分年表
- 自己PR
- 取り組んだこと  
・  
・  
・  
などの機能を含む自己分析ツールを目指す


### コマンド
- backend(Django)環境のコンテナへ入る
docker-compose up -d  
docker-compose exec django /bin/bash  
  
- backend(Django)立ち上げ  
python manage.py runserver 0.0.0.0:8000
